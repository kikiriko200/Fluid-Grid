## Usage

You can use this like Bootstrap.

### class

1. for PC (12 columns)

```
.col-*
```

2. for Tablet (8 columns)

```
.col-m-
```

3. for Smartphone (4 columns)
```
.col-s-
```

4. 960px Grid(recommend to use to PC)
```
.v960
```

5. Trisect a row
```
.col-m-1-3
.col-s-1-3
```
